# Deployment steps

1. Create a new public GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to https://share.streamlit.io and sign in with GitHub.
4. Click Create app / New app.
5. Select your repository, branch `main`, and main file path `streamlit_app.py`.
6. Add secrets in Streamlit Cloud settings if you want News, ChatBot, or Email features.
7. Deploy.

Do not upload `.streamlit/secrets.toml` to GitHub.
